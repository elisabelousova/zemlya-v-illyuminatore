# Telegram Mini App: Земля в иллюминаторе

Мобильный каталог постов канала `cloud_watchers`. Пользователь выбирает тему или страну и переходит к конкретному посту в Telegram.

Полное продуктовое и техническое задание: [docs/TECHNICAL_SPEC.md](docs/TECHNICAL_SPEC.md).

Результаты разбора всей истории канала: [docs/CONTENT_AUDIT.md](docs/CONTENT_AUDIT.md).

## Что уже работает

- четыре раздела: визы, путешествия, экономия (включает акции и распродажи) и пересадки — ровно по лимиту ТЗ;
- блок «Начать отсюда» на главной с редакционной подборкой (флаг `featured: true` у поста);
- фильтры по странам внутри раздела;
- поиск по заголовкам, описаниям и тегам;
- логика актуальности: `lastVerifiedAt` помечает пост «Обновлено» (проверяли недавно) или «Может быть неактуально» (давно не проверяли), `expiresAt` скрывает по-настоящему истёкшие акции и промокоды из обычной выдачи;
- сохранение материалов в Telegram CloudStorage с локальным резервом для браузерной версии;
- открытие постов через Telegram Web App API, с фоллбэком на обычную вкладку вне Telegram;
- фоллбэк на флаг/иконку, если картинка обложки не загрузилась;
- базовые аналитические события в консоль (см. `track()` в `app.js`) — готовы к подключению privacy-friendly счётчика;
- адаптация под мобильный экран и safe area Telegram.

## Как открыть локально

Откройте `index.html` в браузере. Для размещения в Telegram файлы нужно опубликовать на любом HTTPS-хостинге, например GitHub Pages, Cloudflare Pages или Vercel.

## Как добавлять посты

Каталог находится в `data.js`. Для нового материала достаточно добавить объект:

```js
{
  id: "unique-id",
  section: "visa",
  country: "Италия",
  flag: "🇮🇹",
  title: "Название поста",
  description: "Короткое описание",
  url: "https://t.me/cloud_watchers/123",
  tags: ["виза", "италия"]
}
```

Для изменяемого или временного контента (визовые правила, акции, промокоды) можно дополнительно передать в `extra`:

```js
catalogPost(123, "saving", "Европа", "🏷️", "Заголовок", "Описание", ["теги"], {
  lastVerifiedAt: "2026-06-01", // когда последний раз проверяли актуальность
  expiresAt: "2026-07-01",      // когда акция/промокод точно перестают работать
});
```

## Подключение к Telegram

1. Создать бота через `@BotFather`.
2. Опубликовать приложение по HTTPS.
3. В `@BotFather` выбрать `/newapp` и передать URL приложения.
4. Добавить кнопку запуска Mini App в меню бота или закреплённый пост канала.

Токен бота не требуется хранить в этом frontend-проекте.
